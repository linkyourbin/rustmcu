## 🔧 自定义调整

### 修改卡片内容

编辑 `content/_index.md`：

```html
<div class="feature-card">
  <span class="feature-icon">📖</span>
  <h3 class="feature-title">新手入门</h3>
  <p class="feature-desc">从零开始，搭建开发环境</p>
  <a href="/toolchain/" class="feature-link">查看教程</a>
</div>
```

### 修改颜色主题

编辑 `assets/css/extended/home-layout.css`：

```css
/* 搜索并替换 #CE422B 为你的颜色 */
background: linear-gradient(135deg, #CE422B, #ff6b4a);
```

### 调整卡片数量

增加或删除 `.feature-card` 块即可。

### 修改标题

编辑 `hugo.yaml`：

```yaml
homeInfoParams:
  Title: "你的标题"
  Content: |
    你的副标题
    第二行文字
```

## 🐛 问题排查

### 主页显示不正确

**清除缓存**:
1. 按 `Ctrl + Shift + R` 强制刷新
2. 清除浏览器缓存
3. 重启 Hugo 服务器

**检查文件**:
1. 确认 `content/_index.md` 存在
2. 确认 `assets/css/extended/home-layout.css` 存在
3. 检查 `hugo.yaml` 配置正确

### 样式不生效

**检查CSS加载**:
1. 按 F12 打开开发者工具
2. 切换到 Network → CSS
3. 确认 `home-layout.css` 状态码为 200

**检查路径**:
```html
<!-- 在 _index.md 中，CSS路径应为 -->
@import url('/css/extended/home-layout.css');
```

### 响应式问题

**测试不同尺寸**:
1. F12 → Toggle device toolbar
2. 测试 Mobile / Tablet / Desktop
3. 检查网格布局是否正确切换
